Code

For displaying code, the <code>-tag is used. Simply set the desired language as class of the tag and you will get a code block like this:

public class BinarySearch { public int search(int[] list, int item) { int index = Integer.MAX_VALUE; int low = 0, high = list.length - 1; while (low <= high) { //... } return index; } }

To display inline code, simply add the class inline to the <code>-tag. Moreover there are the following options for code elements:

Class Description Result
nohighlight this will only change the font to a monospace font from x import y
plaintext this disables syntax highlighting (the text will have only one color) from x import y
[language-name] if you add the name of the programming language you'll get syntax highlighting from x import y
wrap if necessesary, there is the option to make the code wrap to the next line from x import y as long-name
Note: The < and > characters generally do not have to be replaced with &lt; and &gt; for it to work, because all < characters inside a code element are automatically replaced with JavaScript. The only case where this doesn't work is when you have a <code>-tag inside your code element. Then you'll still have to replace the character yourself, otherwise it's optional.
To change the theme of the syntax highlighting, you have to change the link of the stylesheet in the header for the light mode style and the link in the JavaScript code for the dark mode style.

Links

There are two possible styles for <a>-tags:

HTML Result
<a href="#">Color</a> Color
<a href="#" class="no-color">No color</a> No color
Keep in mind that links (or HTML tags in general) do not work inside <code>-tags.

Note: the <thead> and <tbody> tags are not necessesary, but are considered good style.
