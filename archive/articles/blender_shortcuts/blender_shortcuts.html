Blender Shortcuts

Published: 12.10.2024, last edit: 23.03.2025
Blender 4

Camera and viewport

Ctrl + Alt + NumPad0 - Move the camera to the current view

NumPad, - focus view on the selected object

Edit Mode

Ctrl + E - Open Edge menu

Ctrl + E > Mark Sharp/Clear Sharp - (un)mark an edge as sharp, this affects e.g. shade smooth

Shift + V - Slide selected vertices along edges (pressing G twice also works)

Other

Shift + Tab - Toggle snap on/off (the magnet at the top)

Troubleshooting / quick tips

A single flat color is slightly noisy after exporting to an image

-> Output > Post Processing > set Dither to 0.0 - this disables dithering (color bands may start to show on other image areas)

Make a light only affect some objects

Select the light > Object Properties > Shading > Light Linking - add the collection of objects that the light source should light up
